Data - データ

Data は、機械的に構成したり、手で入力することも、可能ですが、ここでは、ある程度の大きさも持った、実際の Data の分析を目指します。最初は、Data を探し、目的に合わせて利用できる形にする必要があります (Data Wrangling, Munging, or Reshaping)。このプロセスが、もっとも手がかかる部分で、Data Engineering とも呼ばれます。Data Science を学ぶには、まずは、整えられた Data を利用するのが、良いでしょう。その上で、実際の Data と取り組みましょう。

  • R には、付属の Data が付属しており、R の　package にも、Data が付属していることが多いので、まずは、それらを試してみましょう。 Data の集合体が package となっているものもあります。
    • Built-in（R が Install されたときに、base に含まれる）Datasets は、R の Console または　R Script などで、library(help = "datasets") とすると、リストを見ることができます。cars も含まれます。また、ggplot2 など他の package を読み込んだあとで、data() とすると、そのときに利用可能な Data のリストを見ることができます。
    • R の version 4.0.0 の datasets の情報は Statistical Data Analysis に掲載されています。
    • RStudio の 右下の pane の Help tab で、たとえば、cars と入力するか、Console または R Script で、? cars と入力すると、基本情報が得られ、?? cars と入力すると、cars という言葉が含まれる他の package の Data の情報も表示されます。
    • Built-in dataset には Examples がついているものもあり、たとえば example(cars) などで実行できます。ただし、膨大な時間がかかると、注意がついている場合もあります。
    • Built-in Datasets with Explanations and Examples: ? data と、head(data), example(data) の出力をリスとしてあります。

Public Data

Public Data として様々な国際機関や、政府、大学や、研究所などがデータを公開しています。登録が必要な場合もありますが、基本的には、自由に、Download することができます。
データ取得のための、API や、R Package を提供している場合も多いので、調べてみましょう。

まず、Public Data と検索してみてください。商用のサイト以外に、Google Public Data が見つかると思います。日本語で検索すると、日本語サイトが現れ、世界開発銀行の世界開発指標へのリンクなど、いくつかのサイトのリストがあらわれます。つぎに、言語を英語に切り替えると、長いリストが現れます。言語の切り替え方法が不明な場合は、下のリンクをクリックしてください。
